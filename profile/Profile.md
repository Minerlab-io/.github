# Minerlab

   [![Website](https://img.shields.io/badge/website-minerlab.io-blue?style=for-the-badge)](https://www.minerlab.io)
   [![Discord](https://img.shields.io/badge/discord-minerlab-5865F2?style=for-the-badge&logo=discord&logoColor=white)](https://discord.gg/minerlab)

   Minerlab builds infrastructure and tooling for crypto mining and node operations.
   This organization hosts our internal codebases for:

   - Mining infrastructure
   - Node management
   - Miner software and tooling

   All repositories are private and intended for Minerlab engineers and collaborators.

   ---

   ## Focus Areas

   ### Mining Infrastructure

   We maintain a mining monorepo that includes:

   - Miner and pool components
   - Deployment and operations tooling
   - Monitoring, alerting, and basic automation

   The goal is to make it straightforward to run and operate mining setups in a repeatable way.

   ### Node Management (NodeOP / Litenode)

   Our nodeâ€‘oriented projects focus on:

   - Running and monitoring nodes for supported chains
   - Managing node fleets (provisioning, updates, health checks)
   - Handling uptime, synchronization, and basic protection at the infrastructure level

   These repositories provide the building blocks for running nodes in a more controlled and observable way.

   ### Miner Software & Tools

   We also work on:

   - Miner clients and related tooling
   - Management utilities for configuring and updating miners
   - Integrations between miners, pools, and node infrastructure

   Most of this code is tailored to our own environments and operational requirements.
